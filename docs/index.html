Cybersecurity Toolbox for Linux

ZZYUTIL is a terminal-based (TUI) cybersecurity toolkit written in Rust. It bundles 50+ security tools organized into categories like reconnaissance, enumeration, exploitation, web testing, and post-exploitation — all accessible from a single interface.

Install

git clone https://github.com/0xClumzzy/zzyutil.git
cd zzyutil
cargo build --release
./target/release/zzyutil

Features

  • 50+ pentesting tools organized by category
  • Beautiful TUI with 3 themes (dark, compatible, cyber)
  • Auto-install missing tools via pacman, apt, paru, yay
  • Interactive tool execution with live output
  • Search and filter tools by name
  • Show only installed tools
  • Multi-select and batch execution
  • Built with Rust — fast and lightweight

Tool Categories

  • Reconnaissance — nmap, amass, subfinder, masscan, rustscan
  • Enumeration — gobuster, ffuf, enum4linux, nikto, smbclient
  • Exploitation — metasploit, sqlmap, commix, searchsploit
  • Web — httpx, nuclei, dirb, wpscan, sslyze
  • Password Attacks — john, hashcat, hydra, medusa
  • Post-Exploitation — ligolo-ng, lazagne, pspy, chisel
  • Forensics — binwalk, exiftool, steghide, sleuthkit
  • Wireless — aircrack-ng, wifite, reaver, kismet
  • Reverse Engineering — radare2, gdb, strace, ltrace
  • Privilege Escalation — pspy, linux-exploit-suggester
